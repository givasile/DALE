Description of the provided scripts:

DALE implementation:
- feature_effect: contains the implementation of DALE

Examples used in the paper:
- dale_paper_OOD.py: figures/results used in section 4.3
- dale_paper_synthetic_dataset_efficiency.py: figures/results used in section 5.1 (Case 1)
- dale_paper_synthetic_dataset_accuracy.py: figures/results used in section 5.1 (Case 2)
- dale_paper_real_dataset_1_efficiency.py: figures/results used in section 5.2 (for efficiency comparisons)
- dale_paper_real_dataset_1_accuracy.py: figures/results used in section 5.2 (for accuracy comparisons)

Data:
- data: directory with the Bike-Sharing dataset


requirements.txt: txt file with all necessary python packages for running the scripts

Python version 3.9 is required!
